# Project-Delivery
Sistema para gerenciamento e acompanhamento em tempo real da sua entrega.

## Aplicaçao Motoboy

#### Descriçao:
O Sistema vai ter como finalidade prover um serviço para motoboys e clientes que queiram usufluir de seus
serviços de tele-entrega.

### Como funciona:

#### Motoboy

Motoboy vai se cadastrar atraves do site ou do aplicativo mobile, preenchendo campos minimos para ser
incluido no sistema.
- Nome completo
- CPF / CNPJ
- Celular Para contato
- Cidade em que atuara


#### Cliente

Cliente vai ter que fazer um cadastro minimo tambem para afirmar sua autenticidade.
- Nome completo
- CPF / CNPJ
- Cidade que reside
